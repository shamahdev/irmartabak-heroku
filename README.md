# Web Insinyur Martabak on Development
![Insinyur Martabak Logo](static/img/icon.png)

## Project setup
```
npm install
pip install requirements.txt
```

### Bundle Webpack (need to run every change made)
```
npm run build
```

### Start Server
```
open terminal then run server.bat
npm start
```
